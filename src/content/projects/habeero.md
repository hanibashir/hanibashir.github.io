### About the Project

***Habeero Online Shop***  
Habeero is an e-commerce project where I gained hands-on experience in managing online stores, administering Linux servers, deploying web applications, and building automation workflows using self-hosted tools.

### Key Achievements:

- ***Online Store Management:***
  Managed a WooCommerce-based online store, including product catalogs, payment gateways, shipping methods, customer accounts, and order processing.

- ***Server Administration:***
  Configured and maintained Linux servers, domains, SSL certificates, email services, and web hosting environments.

- ***Deployment & Maintenance:***
  Used Git for version control and implemented deployment workflows to streamline application updates and website maintenance.

- ***Automation Workflows:***
  Built and tested automation workflows using n8n and Coolify to automate operational tasks and improve efficiency.
